# Palavras Reservadas

## Definição

Palavras com significado fixo em uma linguagem (ou pseudocódigo) — não podem ser usadas para outros fins. Cada uma tem uma **sintaxe** que define os elementos obrigatórios e opcionais para compor uma instrução.

São necessárias tanto em pseudocódigo quanto em linguagens de programação reais, para que compiladores e interpretadores possam processar o código.

## Categorias (base: Manzano + Forbellone)

| Categoria | Palavras |
|-----------|----------|
| Estrutura de bloco | INÍCIO, FIM, DECLARE |
| Entrada / Saída | LEIA, ESCREVA |
| Condicional | SE, ENTÃO, SENÃO, FIMSE, CASO, SEJA, FIMCASO |
| Repetição | ENQUANTO, FIMENQUANTO, PARA, FIMPARA, REPITA, FAÇA, ATÉ, PASSO, DE |
| Sub-rotinas | PROCEDIMENTO, FUNÇÃO |
| Orientação a Objetos | CLASSE, ATRIBUTO, MÉTODO |
| Tipos de dados | INTEIRO, REAL, CARACTERE, CONJUNTO |
| Arquivos | ARQUIVO COMPOSTO DE, ABRA, FECHE, GUARDE, COPIE, ELIMINE, FIM_REGISTRO, REGISTRO, TIPO |
